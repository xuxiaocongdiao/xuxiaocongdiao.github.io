#

实验二 Git的使用

一、实验目的

1.掌握git使用

2.掌握使用Github作为远程仓库

二、实验任务

1.练习本地git操作

2.用SSH同步本地与Github仓库

三、实验步骤与结果

1.同步本地与Github仓库：

下载Github客户端并安装成功

打开Github客户端并登录

在git目录下创建一个名为android的项目

将创建好的新的项目同步到Github客户端

在客户端可以看到一个未提交的修改

填写评论并提交修改

点击右上角的sync同步按钮将其提交到Github网站上

在Github网站上可以看到新增的commits，表示已实现同步，重复修改同步即可

2.分支开发与合并：

打开Github客户端，点击左上角的create new branch，填写分支名称music

在android文件中写入一句评论语句并保存

提交修改并同步到Github

在Github客户端打开，点击compare&pull request并提交将该分支合并到master中

在Github网站上可以看到新增的branch，表示已实现同步，重复修改即可

四、实验小结

这样在Github客户端中修改更加的简便，并且可以看到修改的次数，对于随时随地的保存和修改项目有着极大的作用。
